# Insurance Pricing Statistical Analysis

## Project Overview

This project is a Data Science analysis based on the **Medical Cost Personal Dataset**.  
The main goal is to help a health insurance company understand which policyholder factors affect annual insurance charges.

The project focuses on **statistical evidence**, not only graphs.  
It includes probability distributions, hypothesis testing, confidence intervals, effect sizes, and simple business interpretations.

## Dataset

The dataset contains **1,338 policyholders** and 7 columns:

| Column | Description |
|---|---|
| age | Age of the policyholder |
| sex | Male or female |
| bmi | Body Mass Index |
| children | Number of dependants covered |
| smoker | Whether the person is a smoker or not |
| region | Policyholder region |
| charges | Annual medical insurance charges in USD |
